# TODOLIST
# To-Do List Application

A simple yet powerful to-do list web application that allows you to manage tasks with due dates and priority levels.

## Features
- Add tasks with descriptions, due dates, and priority levels (Low/Medium/High)
- Filter tasks by priority level
- Responsive design that works on desktop and mobile devices
- Clean and intuitive user interface

## Installation
1. Clone this repository or download the source files
2. Open `index.html` in your web browser

## Usage
1. Enter your task in the input field
2. Select a due date (optional)
3. Choose a priority level (optional)
4. Click "Add Task" button
5. Use the filter buttons to view tasks by priority
